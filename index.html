Tools & Technologies

In my daily work as a Cybersecurity Specialist, I use a wide range of technological tools to detect vulnerabilities, monitor systems, analyze network traffic, and ensure system integrity. Below are some of the key technologies I master and use in my projects:

Wazuh

Open-source SIEM system for security monitoring, threat detection, incident response, and compliance, capable of analyzing logs and detecting anomalies in real-time.

SIEM

Wireshark

Network protocol analyzer that allows detailed examination of real-time traffic. Used for packet capture, identifying suspicious communications, and network forensics.

Network Analysis

Nmap

Essential tool for network scanning and security auditing, enabling discovery of hosts, services, and vulnerabilities. Crucial for infrastructure mapping.

Network Scanning

Nginx

High-performance web server and reverse proxy, used for secure configurations, WAF implementation, and enhancing web application perimeter security.

Infrastructure

PostgreSQL

Relational database management system configured with strong security measures, encryption, and auditing. Implement database hardening to protect critical information.

Database Security

Kali Linux

Specialized Linux distribution for security, including hundreds of tools for penetration testing, digital forensics, and ethical hacking. My primary platform for security audits.

Operating System

Metasploit

Penetration testing framework that facilitates exploit development and execution against remote systems. Fundamental for validating vulnerabilities in controlled environments.

Penetration Testing

Burp Suite

Integrated platform for web application security testing. Includes features like an intercepting proxy, spider, scanner, and repeater for identifying vulnerabilities.

Web Security

OpenVAS

Comprehensive vulnerability assessment framework providing scheduled scans and detailed reports. Implemented for continuous infrastructure monitoring.

Vulnerability Analysis

OWASP ZAP

Penetration testing tool for finding vulnerabilities in web applications. Used to identify issues like SQL injection, XSS, and CSRF.

Application Security

Splunk

Platform for big data analysis and security monitoring, allowing event correlation, alert generation, and visualization of threat patterns from large log volumes.

Data Analysis

AWS Security Hub

Cloud security service providing a comprehensive view of security status in AWS. Implemented to secure cloud deployments and maintain compliance.

Cloud Security
